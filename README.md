# Flashcards for Even Realities G2 (via Even Hub)

Flashcard app for Even Realities G2 glasses with FSRS scheduling.

## Build

```sh
$ npm install
$ npm run build
```

## Development (Simulator)

```sh
$ npm run dev
```

And then in a different terminal:

```sh
$ npm run simulator
```

## Tests

```sh
$ npm run test
```
